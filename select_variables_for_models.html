Grass/PDO model selection

Erica Christensen

5/12/2022

What components of climate are best correlated to perennial grass cover 1916-1979? We start with a list of climate variables that vary at different time scales: short-term variables that vary year to year, El Nino Southern Oscillation (ENSO)-related variables that vary on a 5-7 year time scale, and Pacific Decadal Oscillation (PDO)-related variables that vary on a 30-40 year time scale. First we select variables that are independent from each other and have the best correlative values with grass cover, then test variables in linear models.

Select short-term variables

Short-term climate variables that are expected to affect perennial grasses: total yearly precipitation, total summer (growing season) precipitation, total winter (non-growing season) precipitation, yearly average Palmer Drought Severity Index (PDSI), summer average PDSI, yearly average Standardized Precipitation-Evapotranspiration Index (SPEI), summer average SPEI, summer De Martonne index, and mean annual temperature.

We looked at cross-correlation plots to determine if lags should be considered.

A lag of 1 may be important for yearly precip, PDSI, SPEI, and summer PDSI. We then looked at Pearson’s correlation between each variable and grass to determine if a relationship likely exists. We also looked at scatterplots to determine if a non-linear relationship was needed.

There is no evidence that non-linear relationships need to be included. There is poor correlation between grass and De Martonne, temperature, summer ppt, summer spei, winter ppt, and yearly ppt so these were removed from consideration. Finally, we looked for correlations among remaining variables.

PDSI and SPEI are highly correlated. We chose to proceed with variables PDSI and PDSI lag1, as these variables had the highest correlation with grass and are not correlated with each other.

Select ENSO variables

We repeated the above process with several ENSO-related variables: Nino 3.4 index, Oceanic Nino Index (ONI), and the categorical variable of ENSO phase (each year is categorized as El Nino, La Nina, or Neutral).

We looked at cross-correlation plots to determine if lags should be considered.

No lags need be considered for these variables. Next we looked at Pearson’s correlation and scatterplots

Select PDO variables

We repeated the above process to determine the significance of the PDO index and PDO phase (categorical).

We first looked at cross-correlation plots to determine if lags should be considered.

Lag of 0-10 appear to be significant for PDO index. Rather than creating a new variable for each lag, we created a 10-year moving average smooth of the PDO index.

We next looked at Pearson’s correlation and scatterplots.
